# 《Android / TypeScript / Node.js 面试完全指南（2026）》

> 从移动端到全栈，系统性面试准备手册

---

## 关于本书

本书是为 **Android + TypeScript + Node.js 技术栈**的开发者编写的系统性面试准备指南。

无论你是准备跳槽到外企、冲击大厂，还是寻找远程工作机会，本书都能帮你系统性地查漏补缺。每道面试题都包含**考察要点**和**详细参考答案**，系统设计题配有完整的架构图和方案分析。

**目标读者**：有 2+ 年开发经验，技术栈涵盖 Android（Kotlin）、TypeScript/React、Node.js 中至少一个方向的工程师。

- 寻找远程工作机会、需要英文面试准备的候选人
- 想从单一方向拓展到全栈的开发者

## 使用建议

1. **按方向精读**：先集中复习你的主力方向（Android / 前端 / Node.js）
2. **系统设计必练**：14 个系统设计案例覆盖高频考点，建议手画架构图练习
3. **算法每日一题**：第 15 章的 50 题按 30 天安排，每天 1-2 题
4. **行为面试别忽视**：外企尤其重视，提前准备 STAR 故事
