\chapter*{前言}
\addcontentsline{toc}{chapter}{前言}

本书面向\textbf{有 Python 基础、需要在短时间内冲刺面试}的读者。它不是 PyTorch 大部头教程，
而是一份\textbf{尽量简单、精炼、高频}的速通指南：只讲面试真考、实战够用的内容，
帮你\textbf{快速入门到能应付面试}。

\textbf{怎么用这本书：}
\begin{itemize}[leftmargin=2em]
  \item 时间紧：直接看每章末的「面试速记卡」，再扫一遍第 4、5、7 章的高频题。
  \item 有几天：从第 1 章顺序读到第 8 章，跟着敲一遍最小可跑代码。
  \item 面试前一晚：只翻第 7 章（速查表+坑）和各章速记卡。
\end{itemize}

\textbf{深度取舍：}砍掉了论文复现、手写 autograd 引擎等深水区；但保留了 LLM 时代\textbf{面试高频的分布式训练}
（DDP / FSDP / 并行策略，见第 6 章）。想深入 AI Agent 应用层，可参阅同仓库的《AI Agent 工程师转行与面试完全指南》。

\vspace{0.5em}
\noindent\textit{代码基于 PyTorch 2.x 现代 API，力求可运行、可复现。}
